# Nuxt UI Editor + Jazz Real Time Collaboration Demo

To use this project:

1. Clone the repository
2. Run `npm install`
3. Get a Jazz API key from [jazz.tools](https://jazz.tools)
4. Set the Jazz API key in the `.env` file as `NUXT_PUBLIC_JAZZ_API_KEY`
5. Run `npm run dev`
6. Open the browser and go to `http://localhost:3000`
